##### If Condiiton With Logical Operators:
- And:
```python
a = 0
if a > 0 and a % 2 == 0:
    print('A is an even number and positive integer)
```

- Or:
```python
user = 'anu'
access_level = 3

if user == 'admin' or access_level >= 4:
    print('Access Granted')
else:
    print('Access Denied')
```
